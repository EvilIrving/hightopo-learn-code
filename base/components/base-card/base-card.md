# 基础卡片

## API

### Props

- layout 布局
  - 'square-s' 正方形小 70px*70px
  - 'row' 一行 160px*40px
  - 'vertical' 垂直 70px*160px
  - 'horizontal' 水平 160px*70px
  - 'square-l' 正方形大 160px*160px

- 背景色
  - backgroundColor
- 圆角
  - borderRadius
- 边框
  - borderColor
  - borderWidth
- 阴影
  - shadow

### Emits

<!-- - layout-change
  - payload
    - layout: 布局 -->

### Methods

- setLayout 设置布局
  - 参数
    - layout: 布局
  - 功能
    - 根据 layout 参数修改卡片大小和需要展示的内容
